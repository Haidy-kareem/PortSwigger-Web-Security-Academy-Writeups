# PortSwigger Web Security Academy Write-ups

This repository contains write-ups for PortSwigger Web Security Academy labs, documenting my journey of learning web security and sharing knowledge with others who are interested in cybersecurity.

Each write-up explains the vulnerability, how it was identified, the exploitation process, and the security concepts behind it to help others understand and learn from these labs.


## Purpose

The goal of this repository is to provide clear explanations and learning resources for beginners in web security, while documenting practical experience with different vulnerability classes.
